XML and JavaScript

What is XML?

XML stands for eXtensible Markup Language much like HTML.

How XML is used?

Designed to store and transport data. It was also designed to be self-descriptive.

What does XML look like?

Elements are arranged in a hierarchy, meaning that elements can contain other elements.

What is JavaScript

It is a scripting language enabling you to create dynamically update content, control multimedia, animate images and etc.

How is JavaScript related to HTML and CSS

HTML defines the structure of your content, CSS determines the style and layout, and JavaScript makes the content interactive

Properties of JavaScript

Addition-It can add new objects by simply giving values to those new objects. Deletion-It uses delete keyword, to delete a property from an object.
