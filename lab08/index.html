Different Types of Form Elements

Input

This is used to create interactive controls for web-based forms in order to accept data from the user. An input element can be type text, password, checkbox, and etc.

Textarea

This is often used in a form, to collect user inputs like comments or reviews.

Fieldset

This element is used to group controls as well as labels within a web form.

Optgroup

This element is used to group related options in a drop down list.

Radio Buttons

This element is used to choose a single element among a group. To allow window to choose a single radio.
